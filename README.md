# illnesses_helper
Программа для помощи с лечения болезней
1.Бот для информации, пользователь пишет название болезни, а бот отвечает как лечить эту болезнь(сыллка на препараты,ближайшая клиника)
2.Парсер, телебот POSTMAN(скачать)
